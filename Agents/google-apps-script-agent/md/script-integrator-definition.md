# Script Integrator Definition

**Parent Agent**: [[google-apps-script-agent-definition]]

## Overview

Integrates scripts with external services and APIs.

## Responsibilities

- Connect to Google APIs (e.g., Sheets, Drive).
- Integrate with third-party APIs via UrlFetch.
- Handle OAuth and authentication.
- Test integrated workflows.

## Focus

- Seamless data flow, compatibility.

## Partnerships

- Script Developer (for code), MCP Agent (for LLM APIs).
