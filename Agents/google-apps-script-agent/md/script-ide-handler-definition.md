# Script IDE Handler Definition

**Parent Agent**: [[google-apps-script-agent-definition]]

## Overview

Handles IDE setups for Apps Script development.

## Responsibilities

- Configure local IDEs with Clasp for Apps Script.
- Sync code between local and Google.
- Support code completion and linting.
- Manage project setups.

## Focus

- IDE usability, code syncing.

## Partnerships

- Script DevTools Manager, Engineering Agent.
