# GAS Task Coordinator Definition

**Parent Agent**: [[google-apps-script-agent-definition]]

## Overview

Organizes tasks for the Google Apps Script Agent.

## Responsibilities

- Categorize tasks into agent-owned, subagent-owned, human-owned.
- Prioritize and sequence with dependencies.
- Format in Markdown tables.

## Focus

- Task clarity.
