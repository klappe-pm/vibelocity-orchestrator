## Important File Patterns
### Python Files
- `model-orchestrator*.py`: Core orchestration logic
- `*_api.py`: Provider-specific API clients
- `*cli.py`: Command-line interfaces  
- `test_*.py`: Test suites
- `zen_*.py`: Zen MCP integration

### Supporting Configuration
- `requirements.txt`: Python dependencies
- `grok-models-config.yaml`: Grok model definitions
- `setup_orchestrator.sh`: Automated setup script
- `start-ollama.sh`: Ollama service startup and management
- `warmup-models.sh`: Local model warmup for better performance

### Documentation
- `orchestrator-readme.md`: Comprehensive system documentation
- `local-models-setup.md`: Complete local models setup guide
- `grok-setup.md`: Grok-specific setup instructions
- `CLAUDE.md.md`: Claude configuration (critical to read)
